Some important of forms:

action:

method attribute:
help u to specify http method
We have two type of HTTP method:
GET:
  • it send data through url.
  • never use for sensetive data.
  • Mostly used for query etc
POST:
  • it send data to server securely.
  • it has no size limitations.
  • most secure than get
